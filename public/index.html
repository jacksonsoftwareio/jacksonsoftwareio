JNote

A plain text multipurpose notepad with superpowers.
Combining word processing, note-taking, project management and personal task management in an easy-to-use interface.
Enhanced by features like markup, followups, crawl searching, browser-like navigation, change logging and much more.

Plain and simple

JNote does word processing, but unlike other word processors, JNote handles it in plain text. You can create titles, headlines, lists and much more. Insert table of contents like you normally do in a word processor. Create drawings in plain text with plain text characters. Insert tables and add more rows and columns when you need them.
